## Run Locally

Unzip the Folder shared on Email.
```bash
git clone https://github.com/DavidAmunga/davidamunga-ellotest.git
```

Go to the project directory
```bash
  cd davidamunga-ellotest
```

Install dependencies

```bash
  npm install
```

Start the project

```bash
  npm run dev
```
